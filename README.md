# Multi-Currency Accounting System

## Overview
Developed a Multi-Currency Accounting System with offline-first architecture, supporting real-time synchronization between desktop, mobile, and online databases.

## Features
- Offline-first operation
- Real-time sync between devices
- Works during internet outages
- Automatic sync when connection restores
- Multi-currency support

## Tech Stack
- C# / PHP / MySQL / SQLite (example)
- jQuery / AJAX
- Local + Cloud database sync
